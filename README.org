* Build It

Use any D compiler to compile =main.d=.
#+begin_src sh
  dmd main.d
#+end_src

You can now run it like any other executable.

Linux:
#+begin_src sh
  ./main
#+end_src

Windows:
#+begin_src sh
  main.exe
#+end_src

* Run It

Alternatively, use `rdmd` to run it like a script.
#+begin_src sh
  rdmd main.d
#+end_src
