* Build It

Use any D compiler to compile =main.d=.
#+begin_src sh
  dmd main.d
#+end_src

** Run It

#+begin_src sh
  ./main
#+end_src

Alternatively, if on Windows:
#+begin_src sh
  main.exe
#+end_src

Alternatively alternatively, use `rdmd` to run it like a script.
#+begin_src sh
  rdmd main.d
#+end_src
